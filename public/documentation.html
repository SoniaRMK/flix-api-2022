The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format

Allow new users to register /users POST A JSON object holding data about the user to add, structured like: 
                    
                    {
                    "name": Jane Doe
                    "favoriteMovies":[]
                    }
A JSON object holding data about the user that was added, including an ID: 
                    
                    {
                    "id": 1,
                    "name": Jane Doe
                    "favoriteMovies":[]
                    }
Allow users to update their user info (username) /users/[id] PUT None a text message indicating the user that was updated
Allow users to add a movie to their list of favorites /users/[id]/[movies]/[movieID] POST A JSON object holding data about the movie to add to list of favorites A text message indicating that a movie has been added
Allow users to remove a movie from their list of favorites /users/[id]/[movies]/[movieID] DELETE None A text message indicating that a movie has been deleted
Allow existing users to deregister /users/[id] DELETE None A text message indicating whether the user was successfully removed
